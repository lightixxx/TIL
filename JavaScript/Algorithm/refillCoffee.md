# 커피 리필양 계산하기

## 문제설명
마신 커피의 3분의 2만 리필해주는 카페가 있다.
예를 들어 처음 커피를 90ml 주문하면 첫 리필은 60ml를 해주며, 그 다음 리필은 40ml를 해준다. 횟수 제한은 2번이다.

<br />

## 입출력 예
첫 커피양 - 360 / 마신커피 - 760 (360 + 240 + 160)

<br />

## 나의 풀이
```jsx
var 첫커피 = 360;
var 마신커피 = 0;
var 리필커피 = 첫커피 / 3 * 2;

마신커피 = 첫커피 + 리필커피 + 리필커피 / 3 * 2;

console.log(마신커피);
```
