# 이자율 계산하기

## 문제설명
예금액이 5만원 미만이면 이율이 연 15%이고,
예금액이 5만원 이상이면 이율이 연 20%라고 한다.
2년 후의 예금액을 자동으로 계산해주는 코드를 작성하시오.

<br />

## 입출력 예
예금액 - 60000 / 미래예금액 - 62424

<br />

## 나의 풀이
```jsx
var 예금액 = 60000;
var 미래예금액 = 0;

if(예금액 >= 50000) {
  미래예금액 = 예금액 * 1.2 * 1.2
} else {
  미래예금액 = 예금액 * 1.15 * 1.15
}

console.log(미래예금액)
```
