# Math 객체
수학 상수와 함수를 위한 프로퍼티와 메서드를 제공하는 빌트인 객체이다. 자바스크립트의 기본 내장 객체 중 유일하게 생성자 함수를 사용하지 않는 객체이다. 따라서 정적 프로퍼티와 메서드만 제공한다.

<br />

## Math property

<br />

### Math.PI
PI 값(π ≈ 3.141592653589793)을 리턴한다.
```jsx
Math.PI; // 3.141592653589793
```
<br />

## Math Method
Math 객체의 메서드로 넘겨진 모든 매개변수는 종류에 관계없이 모두 숫자로 변환한다.

<br />

### Math.abs(x)
x의 절댓값(0 또는 양수)을 리턴한다.
```jsx
Math.abs(-1); // 1
Math.abs('1'); // 1
Math.abs(''); // 0
Math.abs(null); // 0
Math.abs(undefined); // NaN
Math.abs('lightix'); // NaN
```

### Math.round(x)
인수를 반올림해서 정수를 리턴한다.
```jsx
Math.round(1.234); // 1
Math.round(-1.234); // -1
Math.round(1.678); // 2
Math.round(-1.678); // -2
```

### Math.ceil(x)
인수를 올림해서 정수를 리턴한다.
